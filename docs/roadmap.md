# Roadmap

## Продуктовый слой (что делаем)
- Telegram-бот с подпиской на тематические дайджесты
- Поиск и отбор релевантных постов из TG-каналов
- Уведомления и удобный формат чтения

## Технический слой (как делаем)
- Telethon для парсинга Telegram
- GPT (Groq/GPT-4o-mini) для рерайта и структурирования
- Docker + VPS для деплоя

## Экспериментальный слой (что улучшаем)
- Промпт-инжиниринг для повышения релевантности
- Фильтрация спама и дублей
- Смена модели для удешевления
