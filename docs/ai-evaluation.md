# Оценка качества AI-компонента

**Название проекта:** Topic Digests
**AI-компонент:** Поиск + рерайт постов в тематический дайджест
**Baseline:** Поиск по ключевым словам без ИИ
**Improved version:** GPT с промптом "отбери топ-5 релевантных постов и сделай связный дайджест"

## 1. Метрики

| Категория | Метрика | Целевое значение |
|-----------|---------|------------------|
| **Основные** | Relevance (точность отбора) | ≥ 80% |
| | Success rate (успешных ответов) | ≥ 95% |
| | Latency (время ответа) | ≤ 5 сек |
| | Cost per request | ≤ $0.01 |
| **Дополнительные** | Hallucinations (выдумка фактов) | ≤ 10% |
| | Usefulness (оценка юзером) | ≥ 4/5 |

## 2. Eval-set

| Параметр | Значение |
|----------|----------|
| Количество примеров | 50 постов |
| Источник | 3 TG-канала (технологии, наука, IT-новости) |
| Сценарии | Короткие новости, аналитика, длинные статьи, посты-мемы |
| Сложные случаи | Спам, дубли, устаревшие новости |

## 3. Сравнение с baseline

| Метрика | Baseline | Improved | Изменение |
|---------|----------|----------|-----------|
| **Relevance** | 60% | 85% | **+25%** ✅ |
| **Latency** | 0.5 сек | 2.5 сек | ⚠️ +2 сек |
| **Cost** | $0 | $0.003 | ❌ +$0.003 |
| **Usefulness** | 3.0/5 | 4.5/5 | **+50%** ✅ |

## 4. Качественные примеры

**Пример 1 (хорошо)**
- *Запрос:* /digest нейросети
- *Baseline:* Найдены 10 постов со словом "нейросеть", включая рекламу
- *Improved:* Отобраны 5 реально полезных постов: про выход GPT-5, Sora, Llama 3

**Пример 2 (плохо)**
- *Improved:* Иногда включает нерелевантные посты, если ключевое слово встречается в комментарии

## 5. Вывод

**Что реально улучшилось?**
- Релевантность дайджестов +25%
- Исключена реклама и спам
- Тексты стали читаемыми и структурированными

**За счёт чего?**
- Промпт с инструкцией "отбирай только полезное, исключай рекламу"
- Few-shot примеры хороших и плохих постов

**Что ещё остаётся проблемой?**
- Латенси (2.5 сек против 0.5 у baseline)
- Редкие галлюцинации (5-10% постов)
