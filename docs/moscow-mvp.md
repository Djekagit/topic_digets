| Категория | Функция | Описание | Кто делает |
|-----------|---------|----------|-------------|
| **Must (MVP)** | 1. Парсинг TG-каналов | Сбор постов из выбранных каналов через Telethon | Backend |
| | 2. Поиск по ключевым словам | Фильтрация постов по теме пользователя | Backend |
| | 3. ИИ-дайджест | GPT собирает топ-5 постов в читаемый формат | Backend |

| **Won't now** | 1. Свой поиск по всем TG | Только по заданным каналам | — |

## MVP (минимальный набор функций для демо)

1. Пользователь пишет боту `/digest AI`
2. Бот парсит 3-5 предустановленных TG-каналов за последние 24 часа
3. Бот отправляет дайджест из 5 лучших постов (короткий пересказ + ссылка)
4. Время ответа — не более 5 секунд
